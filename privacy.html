Privacy Policy

CentCom QuickBooks Integration
Last updated: March 17, 2026

Overview

This application is a personal, internal-use integration that connects to QuickBooks Online via the Intuit API. It is not a publicly available product or service. It is operated solely by Aaron Lavee for personal financial management.

Data Collected

This application accesses QuickBooks Online financial data through Intuit's API, including but not limited to: accounts, invoices, bills, vendors, customers, employees, estimates, items, journal entries, purchases, and bill payments. No data is collected from any other source.

How Data Is Used

Data retrieved from QuickBooks Online is used exclusively for personal bookkeeping, financial reporting, and account management. Data is accessed in real time and is not stored in a persistent database.

Data Storage and Security

  • OAuth tokens are stored locally on the operator's machine in an environment file with restricted file permissions.
  • No QuickBooks financial data is persisted to disk or any external database.
  • No data is transmitted to any third-party service, cloud provider, or analytics platform.
  • The application runs on a local machine secured with macOS system-level protections.

Third-Party Sharing

This application does not share, sell, or transmit any data to third parties. All data remains on the local machine of the sole operator.

Data Retention

No QuickBooks data is retained beyond the duration of a single API request/response cycle. OAuth refresh tokens are retained locally until revoked or expired.

User Rights

As the sole user of this application, the operator may revoke access at any time by disconnecting the application from QuickBooks Online via the Intuit developer portal or the QuickBooks Online settings page.
